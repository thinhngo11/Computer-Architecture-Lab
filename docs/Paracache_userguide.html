🧠 Hướng Dẫn Sử Dụng ParaCache

🔗 Truy cập ParaCache

  • Trang web: https://personal.ntu.edu.sg/smitha/ParaCache/Paracache/start.html
  • Không cần đăng nhập – sử dụng trực tiếp trên trình duyệt.

🛠️ Các chế độ mô phỏng

  • Direct Mapped Cache – mô phỏng bộ nhớ đệm ánh xạ trực tiếp
  • Fully Associative Cache – mô phỏng bộ nhớ đệm liên kết đầy đủ
  • 2-Way Set Associative Cache – mô phỏng bộ nhớ đệm liên kết theo tập 2 chiều
  • 4-Way Set Associative Cache – mô phỏng bộ nhớ đệm liên kết theo tập 4 chiều
  • Cache Type Analysis – phân tích hiệu suất các loại cache
  • Virtual Memory – mô phỏng bộ nhớ ảo

📌 Hướng dẫn sử dụng

  1. Chọn chế độ mô phỏng phù hợp từ menu chính.
  2. Nhập địa chỉ bộ nhớ hoặc chuỗi truy cập bộ nhớ.
  3. Quan sát cách dữ liệu được ánh xạ vào cache hoặc bộ nhớ ảo.
  4. Phân tích hit/miss, block replacement, và hiệu suất tổng thể.

💡 Ứng dụng trong Kiến trúc Máy tính

  • Hiểu rõ cách hoạt động của các loại cache trong CPU.
  • Phân tích hiệu suất truy cập bộ nhớ và tối ưu hóa chương trình.
  • Mô phỏng bộ nhớ ảo và quản lý trang (paging).
  • So sánh các chiến lược ánh xạ và thay thế block.
