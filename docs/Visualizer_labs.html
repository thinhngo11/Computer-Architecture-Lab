🧠 Thực Hành MIPS Visualizer

🔗 Truy cập MIPS Visualizer

  • Trang web: MIPS Visualizer
  • Không cần đăng nhập – sử dụng trực tiếp trên trình duyệt.

🛠️ Hướng dẫn thực hành

Thực hiện từng lệnh MIPS sau trong MIPS Visualizer:

  • add – cộng hai thanh ghi
  • addi – cộng với hằng số
  • slt – so sánh nhỏ hơn
  • sub – trừ hai thanh ghi
  • bne – nhảy nếu khác
  • beq – nhảy nếu bằng
  • sw – lưu vào bộ nhớ
  • lw – tải từ bộ nhớ

Yêu cầu: Quan sát và mô tả luồng điều khiển và luồng dữ liệu qua các thành phần phần cứng MIPS như PC, ALU, Register File, Memory, Control Unit.

📝 Bài tập trắc nghiệm

1. Lệnh add: Lệnh này sử dụng thành phần nào để thực hiện phép cộng?

  • A. ALU
  • B. Register File
  • C. Memory
  • D. Control Unit

2. Lệnh addi: Giá trị hằng được đưa vào từ đâu?

  • A. ALU

  • D. PC

  • B. Register File

4. Lệnh sub: Thành phần nào thực hiện phép trừ?

  • D. Register File

5. Lệnh bne: Nếu hai thanh ghi khác nhau, điều gì xảy ra?

  • A. PC được cập nhật theo offset
  • B. ALU lưu kết quả
  • C. Bộ nhớ được ghi

6. Lệnh beq: Điều kiện nhảy được kiểm tra bởi?

  • A. ALU

  • D. Register File

7. Lệnh sw: Dữ liệu được lưu vào đâu?

  • A. Register File

8. Lệnh lw: Dữ liệu được tải từ đâu?

  • A. Register File

  • D. Control Unit
