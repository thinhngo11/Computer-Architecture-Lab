🧠 Thực Hành Compiler Explorer: Chuyển Đổi C → MIPS

🔗 Truy cập Compiler Explorer

  • Trang web: https://godbolt.org
  • Chọn ngôn ngữ: C
  • Chọn trình biên dịch: GCC hoặc Clang
  • Chọn kiến trúc: MIPS32

🛠️ Các bước thực hành

  1. Tạo ra trình C từ đoạn mã C và nhập vào khung bên trái
  2. Quan sát mã hợp ngữ MIPS được tạo ở khung bên phải
  3. Phân tích luồng dữ liệu và điều khiển
  4. Sao chép mã MIPS và chạy trên JsSpim

📌 Các phần thực hành

1. Khai báo biến và toán tử cộng/trừ

int a = 5; int b = 3; int c = a + b - 2;

Câu hỏi: Lệnh nào trong MIPS thực hiện phép cộng?

  • A. addi
  • B. add
  • C. sub
  • D. lw

2. Cấu trúc if/else

if (a > b) { c = a; } else { c = b; }

Câu hỏi: Lệnh nào kiểm tra điều kiện trong MIPS?

  • A. beq
  • B. bne
  • C. slt
  • D. j

3. Vòng lặp

for (int i = 0; i < 5; i++) { sum += i; }

Câu hỏi: Lệnh nào dùng để nhảy về đầu vòng lặp?

  • A. j
  • B. beq
  • C. add
  • D. sw

4. Hàm

int square(int x) { return x * x; }

Câu hỏi: Thanh ghi nào lưu địa chỉ trả về khi gọi hàm?

  • A. $t0
  • B. $ra
  • C. $sp
  • D. $a0

5. scanf và printf

scanf("%d", &a); printf("%d", a);

Câu hỏi: Lệnh nào trong MIPS dùng để tương tác với I/O?

  • A. syscall
  • B. lw
  • C. sw
  • D. j

🧩 Ví dụ tổng hợp

int a, b, c;
scanf("%d %d", &a, &b);
if (a > b) { c = a - b; }
else { c = a + b; }
for (int i = 0; i < c; i++) { printf("%d\\n", i); }
int result = square(c);
printf("Result: %d\\n", result);

Câu hỏi: Lệnh MIPS nào tương ứng với c = a - b;?

  • A. add
  • B. sub
  • C. slt
  • D. syscall

Câu hỏi: Lệnh MIPS nào tương ứng với for (int i = 0; i < c; i++)?

  • A. beq
  • B. j
  • C. slt + bne
  • D. syscall

Câu hỏi: Lệnh MIPS nào tương ứng với printf("Result: %d\\n", result);?

  • A. syscall
  • B. sw
  • C. add
  • D. jr
