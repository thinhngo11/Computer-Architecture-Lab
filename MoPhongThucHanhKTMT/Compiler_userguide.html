🧠 Hướng Dẫn Sử Dụng Compiler Explorer

🔗 Truy cập Compiler Explorer

  • Truy cập trang web: https://godbolt.org
  • Không cần đăng nhập – bạn có thể sử dụng trực tiếp trên trình duyệt.

🛠️ Các bước cơ bản để sử dụng

  1. Nhập mã nguồn
    • Gõ mã C/C++ vào khung bên trái.
    • Ví dụ: int main() { return 0; }
  2. Chọn trình biên dịch
    • Chọn phiên bản trình biên dịch MIPS từ danh sách bên phải (ví dụ: mips gcc 15.2.0).
    • Có thể tùy chỉnh cờ biên dịch như -O2, -Wall, v.v.
  3. Xem mã hợp ngữ
    • Kết quả biên dịch sẽ hiển thị ở khung bên phải dưới dạng mã hợp ngữ.
    • Giúp hiểu cách mã nguồn được chuyển thành lệnh máy.
  4. So sánh trình biên dịch
    • Thêm nhiều khung để so sánh kết quả giữa các trình biên dịch khác nhau.
    • Phân tích hiệu suất và tối ưu hóa mã.

💡 Ứng dụng trong Kiến trúc Máy tính

  • Phân tích cách lệnh C được chuyển thành hợp ngữ MIPS hoặc x86.
  • Hiểu rõ vai trò của trình biên dịch trong quá trình dịch mã.
  • So sánh hiệu quả giữa các trình biên dịch và cờ tối ưu hóa.
  • Học cách viết mã hiệu quả hơn dựa trên kết quả hợp ngữ.

📁 Lưu và chia sẻ

  • Nhấn biểu tượng “Share” để tạo liên kết chia sẻ mã và kết quả biên dịch.
  • Liên kết có thể được gửi qua email hoặc nhúng vào tài liệu học tập.
